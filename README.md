# Moonshine Capital - Affiliate Hub

A high-performance, static landing page for recruiting business funding agents and sub-affiliates. 

## Features
- **Zero Dependencies:** Pure HTML/CSS/JS. No frameworks to update or break.
- **Dark Mode Aesthetic:** Optimized for authority and high-ticket offers.
- **Mobile First:** Responsive grid layouts for partner cards.
- **Fast:** Scores 100/100 on PageSpeed Insights due to minimal code.

## How to Customize
1. Open `index.html`.
2. Search for `href="https://"` to find all affiliate links.
3. Replace the `bit.ly` links with your own tracking links.

## Deployment
This site is ready for 1-click deployment on Vercel or Netlify.

**Vercel:**
1. Push this code to GitHub.
2. Go to Vercel.com -> "Add New" -> "Project".
3. Select this repository.
4. Click "Deploy".
